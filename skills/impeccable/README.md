```bash
npx mdskills install pbakaus/impeccable
```

## 7 大设计领域

1. **Typography（排版）** — 字体搭配、行高、层级
2. **Color & Contrast（色彩与对比度）** — 调色板、WCAG 对比度
3. **Spatial Design（空间设计）** — 间距、网格、对齐
4. **Motion Design（动效设计）** — 缓动曲线、stagger、reduced-motion
5. **Interaction Design（交互设计）** — hover/focus/active 状态
6. **Responsive Design（响应式设计）** — 移动优先、流式布局、Container Query
7. **UX Writing（UX 文案）** — 按钮文本、错误信息、引导文案

| 更多... | 见 SKILL.md |

## 核心理念

- 避免 "AI 味"：不使用泛化的 blue-500、圆角卡片堆砌
- 追求独特性：每个项目有自己的视觉特征
- 可维护性：生成的代码要符合项目既有规范
