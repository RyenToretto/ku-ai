# Skill: find-skills

> 帮助发现和安装新的 Agent Skills。

## 基本信息

| 项目 | 值 |
|------|-----|
| Skill 名称 | `find-skills` |
| 来源 | 预装 |
| 安装位置 | `~/.agents/skills/find-skills/` |
| 触发条件 | 用户问 "how do I do X"、"find a skill for X"、"is there a skill that can..." |
| 依赖 | 无 |

## 功能

- 根据用户描述搜索可用的 Skills
- 推荐匹配的安装命令
- 说明 Skill 的功能和适用场景

## 使用场景

```
有没有什么 skill 可以帮我做 UI 设计？
找一个适合做 TDD 的 skill
```
