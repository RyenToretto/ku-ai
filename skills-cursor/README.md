# Cursor 内置 Skills

Cursor IDE 自带的 Skills，位于 `~/.cursor/skills-cursor/`。

| Skill | 描述 |
|-------|------|
| `babysit` | 维护 PR 状态：分类评论、解决冲突、修复 CI |
| `create-rule` | 创建 Cursor Rules (.mdc 文件) |
| `create-skill` | 创建 Agent Skills (SKILL.md) |
| `create-subagent` | 创建自定义 Subagent |
| `migrate-to-skills` | 将 Rules/Commands 迁移为 Skills 格式 |
| `shell` | 执行 /shell 命令 |
| `update-cursor-settings` | 修改 Cursor/VSCode settings.json |
