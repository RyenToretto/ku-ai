# Lighthouse MCP 使用示例

## 移动端性能审计

```
对 https://www.pixpop.ai 运行 Lighthouse 移动端性能审计，
使用模拟限速，只检查 performance 和 accessibility 类别。
给出评分和前 5 个优化建议。
```

## 完整审计

```
对 https://www.pixpop.ai/explore 运行完整的 Lighthouse 审计（所有类别），
桌面模式，列出每个类别的评分和主要问题。
```

## 优化前后对比

```
1. 先对当前页面运行 Lighthouse 审计，记录分数
2. 应用优化建议
3. 再次运行审计
4. 对比前后分数变化
```
