# Google Stitch MCP 使用示例

## 生成落地页

```
使用 Stitch 生成一个 AI 视频生成工具的落地页：
- 深色主题，主色调 #ff317d（粉色）
- 英雄区域：大标题 + 副标题 + CTA 按钮
- 特性展示：3 列卡片，每张带图标
- 底部 CTA 区域
- 参考 DESIGN.md 中的设计令牌
```

## 从现有网站提取设计

```
1. 用 extract_design_context 提取 https://www.pixpop.ai 的设计系统
2. 生成 DESIGN.md
3. 基于提取的设计系统生成新页面
```

## 多屏幕应用流程

```
生成一个用户注册流程的 5 个关联屏幕：
1. 邮箱输入
2. 验证码输入
3. 设置密码
4. 选择偏好
5. 完成欢迎页

保持一致的设计系统、配色和组件风格。
```

## 获取特定屏幕代码

```
1. list_projects 查看所有项目
2. get_screen_code 获取 "首页" 屏幕的 Tailwind CSS 代码
3. 手动调整为 Vue 组件
```
